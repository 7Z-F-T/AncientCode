矢量图编辑软件Artist Pro v1.0

注意！
-------
请将images文件夹，ap.jar，quaqua.jar放到同一个文件目录下，才能保证程序正常运行！
使用Mac OS look and feel时，菜单中会出现快捷键名称不能正确显示的问题。经验证此问题由于quaqua主题包的缺陷导致，并不是此软件自身的问题。换成其他look and feel后这个问题就不再出现。


使用说明：
------------
双击ap.jar，打开程序

左侧为工具栏，右侧为属性面板（注意有一些属性面板叠在一起，可通过点击上方的标签进行切换）。上方为菜单栏和工具栏。操作的顺序是先选择左侧工具栏中的某种工具，然后绘图。

若当前选中了图形，操作属性面板就可以修改当前选中的对象的各种属性。若当前没有选中任何图形，操作属性面板将修改绘图的默认属性。

绘制文字时，先选择文字功能摁钮，然后在画布任一处点击鼠标，就会出现文字输入对话框，可输入文字。

修改已经输入的文字时，也要选择文字功能摁钮，然后在已有文字内部点击鼠标，对话框重新出现，可以重新修改文字属性。

应用梯度填充时，应先勾选Gradient Paint复选框，然后单击set摁钮，回到画布上，拖出一个梯度填充（类似photoshop）里面的操作。然后再次点击set摁钮，回到原来的编辑模式！请注意再次点击set返回原来的模式很重要！否则系统一直处于绘制梯度填充状态！

保存时，默认的扩展名为.ap
